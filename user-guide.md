# 유저 가이드 (User Guide)

---

## 대시보드 개요

| 항목 | 설명 |
|------|------|
| 총 자산 | 보유 USDT + CNYT 평가액 합산 |
| 월간 수익 | 당월 적립된 수익 |
| CNYT 잔액 | 보유 중인 CNYT 수량 |
| 누적 수익 | 가입 이후 총 수익 합계 |

정산 시간: **매일 UTC 06:00** 자동 입금

---

## 패키지 관리

### 활성 패키지 확인
- **[내 포트폴리오]** → 패키지 목록 · 만기일 · 월간 수익 현황 확인
- Flexible 플랜은 락업 없이 언제든지 원금 회수 가능
- Basic ~ VIP 플랜은 **12개월 락업** — 만기 전 해지 시 원금 **15% 위약금**

### 재투자
- 만기 도래 후 원금을 즉시 재투자하거나 출금 선택 가능
- 상위 플랜 선택 시 CNYT 보상 비율 상승

## 입금 (Deposit)

### 입금 전 체크리스트
- 공식 LONGRISE 사이트 접속 확인
- 2FA 활성화 완료 여부 확인

### 입금 절차
1. **[지갑]** → **[입금]** 클릭
2. USDT 금액 입력 (ERC-20 또는 TRC-20)
3. 플랫폼 전용 지갑 주소를 **세 번 확인** 후 전송
4. 블록체인 확인 후 잔액 반영 (5 ~ 30분)

### 수수료
| 방식 | 수수료 |
|------|--------|
| 플랫폼 입금 수수료 | **0%** |
| 블록체인 네트워크 수수료 | 네트워크 상황에 따라 변동 |

> 잘못된 주소로의 송금은 복구되지 않습니다. 반드시 세 번 확인하세요.

## 출금 (Withdraw)

### 출금 유형

| 유형 | 조건 | 위약금 |
|------|------|--------|
| 수익 출금 | 언제든지 | 없음 |
| 만기 후 원금 출금 | 12개월 만기 후 | 없음 |
| 조기 해지 출금 | 만기 전 (Flexible 제외) | 원금 **15%** |

### 출금 절차
1. **[지갑]** → 가용 잔액 확인 → **[출금]** 클릭
2. 금액 입력 (최소 **$50 USDT**) → 수령 지갑 주소 입력
3. 거래 비밀번호 입력 → 2FA 코드 입력
4. **[출금 확정]** → 배치 처리 대기

### 출금 한도

| 랭크 | 일 한도 | 월 한도 |
|------|--------|--------|
| White | $200 | $5,000 |
| Blue | $500 | $15,000 |
| Purple | $2,000 | $50,000 |
| Red | $5,000 | $200,000 |
| Black | 한도 없음 | 한도 없음 |

## 보안 설정

| 항목 | 설명 |
|------|------|
| 거래 비밀번호 | 모든 금융 거래 시 필수 입력 |
