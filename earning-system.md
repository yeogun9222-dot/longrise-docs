# 수익 시스템 (Earning System)

LONGRISE AI의 전체 수익 구조 — 커미션 · 랭크 · 추천인 프로그램을 통합 안내합니다.

---

## 1. 커미션 구조 (Commission)

### 직접 추천 커미션
내 추천 링크로 가입한 회원이 투자 시 즉시 발생:

- **직접 추천:** 투자금의 **10%**
- **2단계:** 투자금의 **1%**
- **3단계 이하:** 투자금의 **0.1%**
- **최대 25단계**까지 누적 적용 → 총합 **≈ 11.11%**

### 상위 랭크 추가 보상
| 랭크 | 추가 월간 보상 |
|------|-------------|
| Red | 전체 Red 티어 입금액의 **1% / 월** |
| Black | 전체 Black 티어 입금액의 **2% / 월** |

---

## 2. 랭크 시스템 (Rank)

LONGRISE AI는 5단계 멤버십 랭크를 운영합니다:

**White → Blue → Purple → Red → Black**

| 랭크 | 출금 일 한도 | 출금 월 한도 | 주요 혜택 |
|------|-----------|-----------|---------|
| **White** | $200 | $5,000 | 기본 포트폴리오 접근 |
| **Blue** | $500 | $15,000 | 고급 분석 대시보드 · 우선 지원 |
| **Purple** | $2,000 | $50,000 | 전담 계정 매니저 · 주간 인사이트 |
| **Red** | $5,000 | $200,000 | VIP 시그널 · 월 1% 추가 풀 보상 |
| **Black** | 한도 없음 | 한도 없음 | 최고 권한 · 월 2% 추가 풀 보상 |

> 랭크는 조건 충족 시 자동 승급되며, 한 번 달성한 랭크는 하락하지 않습니다.

---

## 3. CNYT 보상

패키지 운용 중 발생하는 수익의 일부를 CNYT 토큰으로 추가 지급합니다:

| 플랜 | CNYT 보상 |
|------|---------|
| Flexible | — |
| Basic | 수익의 **2%** |
| Standard | 수익의 **4%** |
| Premium | 수익의 **6%** |
| VIP | 수익의 **10%** |

**CNYT 토큰 개요:**
- 발행처: Horizon Capital Foundation (HCF)
- 출시가: **$0.01**
- 목표가: **$10.00**
- 기반 자산: $1B+ 규모 아시아 카지노 실물자산 (RWA)
- 기술 표준: ERC-20 · BEP-20
- 거래소 상장 목표: **2026년**

**HCF × LONGRISE:**
HCF는 LONGRISE AI의 전략적 투자자입니다. 이 투자의 일환으로 CNYT 총 발행량의 **20%** 가 LONGRISE 생태계 유저 보상 풀로 배정되었으며, 플랫폼 운용 성과에 따라 순차 지급됩니다.

---

## 4. 추천인 프로그램 (Referral)

### 참여 절차

1. 플랫폼 가입 후 **[내 프로필]** → 고유 추천 링크 확인
2. 링크 공유 → 해당 링크로 가입한 회원의 투자 시 커미션 자동 발생
3. **[대시보드]** → **[추천 현황]** 에서 실시간 통계 확인

### 추적 항목
- 추천 인원 수
- 네트워크 누적 투자금
- 단계별 발생 커미션

### 커미션 수령
- 매일 UTC 06:00 정산 시 가용 잔액에 자동 적립
- 최소 출금 금액: **$50 USDT**
