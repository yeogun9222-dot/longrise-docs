시작하기 (Getting Started)

Step 1 — 계정 생성

  1. LONGRISE 공식 플랫폼 접속
  2. "회원가입" 또는 "JOIN THE EMPIRE" 클릭
  3. 이메일 주소 입력 → 인증 코드 수신 및 입력
  4. 거래 비밀번호 설정
  5. 프로필 정보 입력 (성명 · 휴대폰 · 국가) 후 제출

Step 2 — 보안 설정

  1. [보안 설정] 이동
  2. Google Authenticator 앱 설치 (iOS / Android)
  3. 화면 QR코드 스캔 → 6자리 OTP 확인
  4. 백업 코드 반드시 안전한 곳에 보관
거래 비밀번호와 2FA 코드는 절대 타인에게 공유하지 마세요.

Step 3 — 패키지 선택 및 투자

Step 4 — 입금

  1. [지갑] → [입금] 클릭
  2. 입금할 USDT 금액 입력 (ERC-20 또는 TRC-20)

  • [내 포트폴리오] → 패키지별 운용 현황 및 만기일 확인

수익 시스템 (Earning System)

LONGRISE AI의 전체 수익 구조 — 커미션 · 랭크 · 추천인 프로그램을 통합 안내합니다.

1. 커미션 구조 (Commission)

직접 추천 커미션

  • 직접 추천: 투자금의 10%
  • 2단계: 투자금의 1%
  • 3단계 이하: 투자금의 0.1%
  • 최대 25단계까지 누적 적용 → 총합 ≈ 11.11%

상위 랭크 추가 보상

2. 랭크 시스템 (Rank)

LONGRISE AI는 5단계 멤버십 랭크를 운영합니다: White → Blue → Purple → Red → Black

Confidential by Design

LONGRISE AI의 창립자들은 의도적으로 익명입니다.

이름도, 얼굴도 공개되지 않았습니다. 그러나 아시아 VIP 카지노 씬에서 이들을 모르는 사람은 없습니다. 스스로를 "테이블의 건축가들(The Architects)"이라 부르는 이 4인은, 수십 년의 카지노 운영 경험과 퀀트 트레이딩 노하우를 결합해 LONGRISE를 설계했습니다.

"HCF 창립자 4인 — 비공개"
HCF × LONGRISE 전략적 파트너십
