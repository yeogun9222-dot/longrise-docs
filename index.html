시작하기 (Getting Started)

LONGRISE AI에 가입하고 첫 투자를 시작하는 전체 과정입니다.

Step 1 — 계정 생성

  1. LONGRISE 공식 플랫폼 접속
  2. "회원가입" 또는 "JOIN THE EMPIRE" 클릭
  3. 이메일 주소 입력 → 인증 코드 수신 및 입력
  4. 거래 비밀번호 설정
  5. 프로필 정보 입력 (성명 · 휴대폰 · 국가) 후 제출

Step 2 — 보안 설정

  1. [보안 설정] 이동
  2. Google Authenticator 앱 설치 (iOS / Android)
  3. 화면 QR코드 스캔 → 6자리 OTP 확인
  4. 백업 코드 반드시 안전한 곳에 보관
거래 비밀번호와 2FA 코드는 절대 타인에게 공유하지 마세요.

Step 3 — 패키지 선택 및 투자

12개월 락업 플랜 조기 해지 시 원금 15% 위약금 발생. Flexible 플랜은 락업 없음.

Step 4 — 입금

  1. [지갑] → [입금] 클릭
  2. 입금할 USDT 금액 입력 (ERC-20 또는 TRC-20)
  3. 화면에 표시된 플랫폼 지갑 주소를 세 번 확인 후 전송
  4. 블록체인 확인 후 잔액 반영 (5 ~ 30분 소요)

  • [내 포트폴리오] → 패키지별 운용 현황 및 만기일 확인

수익 시스템 (Earning System)

LONGRISE AI의 5가지 수당 구조 전체를 안내합니다. (V8.9 마스터 플랜 기준)

핵심 보상 5가지 요약

1. 데일리 AI 배팅 수익

VIP 기준 월 수익률 18%~는 아래 3가지 요소의 합산으로 매일 변동합니다.

요소평균 기여설명
Pool Size Factor (쉐어)6%~총 판돈 규모에 따른 하우스 우위
Betting Performance (아바타)9%~4개 거점 베팅 성과 평균
Futures Return (코인 선물거래)3%~AI 코인 선물거래 수익
합계 (VIP 기준)18%~월 평균 (변동폭 14~22%)
30일 통계 (VIP 기준): 평균 17.8% / 최고 21.7% / 최저 14.2%

2. 직접 추천 수당

본인이 직접 추천한 1대의 패키지 구매금액의 10% 즉시 지급. 추가 구매 시 매회 동일 적용.

동일 회원 추가 구매 시에도 구매 건당 매회 10% 지급. 예: 동일 회원이 Premium $1,000을 3회 구매 → $100 × 3 = $300

3. 롤업 수당 (25대 체감형)

직하위 회원들의 배팅 수익금의 10% 매칭, 25대까지 체감 적용.

계산 예시 (하위 회원 월 배팅 수익 $100 기준)
1대: $10 (10%)  |  2대: $1 (10% of $10)  |  3대: $0.10  |  … 25대까지 체감
평균 15대 기준 총합: ≈ 11.11%
⚠️ 직접추천 수당(패키지 구매금액 × 10%)과 롤업 수당(배팅 수익금 × 10%)은 완전히 다른 별개 보상입니다.
