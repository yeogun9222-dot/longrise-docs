# 시작하기 (Getting Started)

---

## Step 1 — 계정 생성

1. LONGRISE 공식 플랫폼 접속
2. **"회원가입"** 또는 **"JOIN THE EMPIRE"** 클릭
3. 이메일 주소 입력 → 인증 코드 수신 및 입력
4. 거래 비밀번호 설정
5. 프로필 정보 입력 (성명 · 휴대폰 · 국가) 후 제출

---

## Step 2 — 보안 설정

1. **[보안 설정]** 이동
2. Google Authenticator 앱 설치 (iOS / Android)
3. 화면 QR코드 스캔 → 6자리 OTP 확인
4. **백업 코드 반드시 안전한 곳에 보관**

> 거래 비밀번호와 2FA 코드는 절대 타인에게 공유하지 마세요.

---

## Step 3 — 패키지 선택 및 투자

LONGRISE AI는 5가지 투자 플랜을 제공합니다:

| 플랜 | 최소 투자금 | 락업 기간 | 월간 수익률 | CNYT 보상 |
|------|-----------|---------|-----------|---------|
| **Flexible** | $100 | 없음 | 4%+ | — |
| **Basic** | $200 | 12개월 | 7%+ | 수익의 2% |
| **Standard** | $500 | 12개월 | 9%+ | 수익의 4% |
| **Premium** | $1,000 | 12개월 | 11%+ | 수익의 6% |
| **VIP** | $5,000 | 12개월 | 18%+ | 수익의 10% |

**투자 확정 절차:**
1. 원하는 패키지 **[구매]** 클릭
2. 투자 상세 정보 및 락업 기간 확인
3. 거래 비밀번호 입력 → 2FA 코드 확인
4. **[투자 확정]** 클릭

> 12개월 락업 플랜 조기 해지 시 **원금 15% 위약금** 발생. Flexible 플랜은 락업 없음.

---

## Step 4 — 입금

1. **[지갑]** → **[입금]** 클릭
2. 입금할 USDT 금액 입력 (ERC-20 또는 TRC-20)
3. 화면에 표시된 플랫폼 지갑 주소를 **세 번 확인** 후 전송
4. 블록체인 확인 후 잔액 반영 (5 ~ 30분 소요)

> 잘못된 주소로의 송금은 복구되지 않습니다.

---

## Step 5 — 수익 확인

- 정산 시간: **매일 UTC 06:00** 자동 입금
- 대시보드에서 누적 수익 · CNYT 보상 · 추천 커미션 실시간 확인
- **[내 포트폴리오]** → 패키지별 운용 현황 및 만기일 확인
